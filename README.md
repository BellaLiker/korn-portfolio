# Simple Portfolio (Next.js)

## 1) Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## 2) Customize
Edit these files:
- `app/page.js` (your content + projects)
- `app/globals.css` (styles)

## 3) Deploy to Vercel
1. Push this folder to a GitHub repo
2. Go to Vercel > New Project > Import repo
3. Framework: **Next.js** (auto-detected)
4. Build command: `next build` (default)
5. Output: default

Tip: You can also deploy directly with Vercel CLI:
```bash
npm i -g vercel
vercel
```
